    Returns Promise<any>

    An the result of the query the same way as the expo-sqlite database does see

    Link

    https://docs.expo.dev/versions/latest/sdk/sqlite/. If, after reading the documentation, you are still unsure what kind of object will be returned, don't worry, that is normal.     Throws

    modifys the sqlite database based on the given query

    Example
